# BukiLauncher
BukiLauncher, basit ve hızlı bir Minecraft başlatıcısıdır.
BukiLauncher ile istediğinin Vanilla ve Fabric sürümlerini indirip oynayabilirsiniz.

## Özellikler
- Kullanıcı dostu arayüz
- Fabric desteği
- Java seçme özelliği

## Kurulum
BukiLauncher'ı depodaki sürüm kısmından indirebilir ve kurulum olmadan kullanabilirsiniz. Ayrıca kaynak kodlarını da indirebilir ve değiştirebilirsiniz.

Eğer BukiLauncher ile ilgili bir sorununuz varsa GitHub sorunu açabilirsiniz.
